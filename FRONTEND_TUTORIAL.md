Below is a series of helpful info and links to tutorials for getting started in web development that I currated for future developers.

HTML
    HTML (Hypertext Markup Language) is the backbone of any website. It's the language used to structure and present content on the web. Here's a helpful video tutorial on HTML for beginners: https://www.youtube.com/watch?v=UB1O30fR-EE

CSS
    CSS (Cascading Style Sheets) is used to style HTML content. It's what makes websites look visually appealing. Here's a helpful video tutorial on CSS for beginners: https://www.youtube.com/watch?v=1PnVor36_40

JavaScript
    JavaScript is used to add interactivity to web pages. It's what makes websites responsive and dynamic. Here's a helpful video tutorial on JavaScript for beginners: https://www.youtube.com/watch?v=PkZNo7MFNFg

CSS Frameworks
    CSS frameworks are pre-built libraries of CSS code that can help you quickly create professional-looking websites. They can save you time and effort, especially if you're working on a large project. Here are a couple of popular CSS frameworks you can learn:

Bootstrap: https://www.youtube.com/watch?v=gqOEoUR5RHg
Materialize: https://www.youtube.com/watch?v=BDpzd1lZV1g

    JavaScript frameworks and libraries are pre-built libraries of JavaScript code that can help you quickly create complex, interactive web applications. Here are a few popular JavaScript frameworks and libraries you can learn:

Responsive Web Design
    Responsive web design is the practice of designing web pages that can adapt to different screen sizes, such as desktops, tablets, and smartphones. Here's a helpful video tutorial on responsive web design: https://www.youtube.com/watch?v=GYKOXkqfVYk

Web Accessibility:
    Web accessibility is the practice of designing web pages that can be used by people with disabilities. It's important to make sure that your websites are accessible to everyone. Here's a helpful video tutorial on web accessibility: https://www.youtube.com/watch?v=gD4f2tJ4Kj4
